# TechCorp Decision Game

TechCorp Decision Game is a turn-based business management simulation written in Java.

The player manages a technology company, hires employees, develops projects, earns revenue, and competes against an AI-controlled company.

## Features

- Turn-based gameplay
- Company management
- Employee system
- Project progress system
- AI competitor
- Random events
- Victory and defeat conditions
- Input validation
- Exception handling
- Unit tests

## Object-Oriented Concepts

This project demonstrates:

- Classes and objects
- Encapsulation
- Inheritance
- Polymorphism
- Interfaces
- Enums
- Exception handling
- Modular design

## Requirements

- Java 17
- Maven

## Run

```bash
mvn clean test
mvn -q -DskipTests package
java -jar target/techcorp-decision-game-1.0-SNAPSHOT.jar
